
# Jumpcore

Jumpcore is a new Jump And Run Plugin for Minecraft Java.


[![MIT License](https://img.shields.io/badge/License-MIT-green.svg)](https://opensource.org/license/mit/)


## Features

- No Database required
- Easy to use
- Made for Paper, Bukkit and Spigot
